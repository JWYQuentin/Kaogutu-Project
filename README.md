# 考古圖 Kaogu Tu — The World's First Archaeological Catalog

A single-page website introducing the *Kaogu Tu* (考古圖), a ten-volume illustrated catalog of ancient Chinese artifacts compiled by Lü Dalin (呂大臨) in 1092 CE during the Northern Song dynasty. Widely recognized as the earliest systematic archaeological catalog in the world, the *Kaogu Tu* documents 210 bronze vessels and 13 jade objects spanning from the Shang dynasty to the Han dynasty.

This project was created as a rare book study for an archaeology course.
